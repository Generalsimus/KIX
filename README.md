```

cd app-name
```
```bash
kix start
```
AND
```bash
kix build
```

## Getting Started

```js
import kix from "kix"
import "./style.scss"

var HTML = <div>
    <h1>KIX :)</h1> 
</div>

kix(document.body, HTML);  
```
